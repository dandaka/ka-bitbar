## About ##

Simple script, that shows Kaliningrad weather station from Baltic bay and forecast from Windguru.

## Install ##

1. Install BitBar
2. Install Node
3. Run `npm install` in this folder
4. Copy .sh script to your BitBar plugins folder
5. Edit .sh script if needed
